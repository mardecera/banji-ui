With your favorite package manager:

```bash

```

### Testing

This library uses **Vitest** and **React Testing Library**.

Run tests with:

```bash
pnpm test
```

### Local Development

Clone the repo and install dependencies:

```bash
git clone https://github.com/mardecera/banji-ui.git
cd banji-ui
pnpm install
```

Start development mode:

```bash
pnpm dev
```

Build the library:

```bash
pnpm build
```
