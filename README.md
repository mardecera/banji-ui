# Banji UI

Banji UI is a React component library for building user interfaces.

## Installation

```bash
pnpm install banji-ui
```

## Usage

```tsx
import { Button } from "banji-ui"

const App = () => {
  return (
    <div>
      <Button>Button</Button>
    </div>
  )
}
```
